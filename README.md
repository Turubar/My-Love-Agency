<h1>Описание проекта</h1>
<p>Это веб-приложение является информационной системой по организации свиданий.
Главная цель - предоставить клиенту подробную информацию об предлагаемых романтических услугах и реализовать возможность оставлять заявку для дальнейшей связи, а также позволить "менеджерам" редактировать данные информационной системы.</p>
<br>
<h2>Стэк</h2>
<p>Backend:</p>
<p>1) ASP.NET Core 6 (MVC)</p>
<p>2) MS SQL Server + Entity Framework Core</p>
<br>
<p>Frontend</p>
<p>1) HTML/CSS/JS</p>
